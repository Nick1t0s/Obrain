# 🧠 Obrain — Умный бот для заметок с ИИ

> Персональный ассистент, который превращает голосовые и текстовые сообщения в структурированные заметки в вашем Obsidian.

```
🎤 Голос → 🧠 Транскрибация → ✨ Очистка LLM → 📝 Obsidian → 📊 Авто-суммаризация
```

---

## 🚀 Возможности

### 📥 Приём заметок
- ✅ Текстовые сообщения
- ✅ Голосовые сообщения (ГС) с транскрибацией через **faster-whisper**
- ✅ Автоматическая очистка текста через **Ollama** (локальные LLM)
- ✅ Сохранение в формате Markdown (Obsidian-совместимый)

### 🗂️ Организация данных
```
📁 Мой мозг/
├── 00_Inbox/raw_data.md          # Сырые заметки (время + текст)
├── 01_Daily/2026-03-27.md        # Обработанные заметки по дням
└── 02_Reviews/
    ├── Weekly/2026-03-24_2026-03-30.md   # Суммари недели (24.03 - 30.03)
    └── Monthly/2026-03.md                 # Суммари месяца
```

### 🤖 Автоматизация
- ⏰ **Ежедневное суммари** — в 23:30 анализирует заметки дня
- 📆 **Еженедельное суммари** — в воскресенье выделяет тренды
- 📊 **Ежемесячное суммари** — в конце месяца строит стратегию
- 🔄 **Catch-up режим** — если бот был выключен, догоняет пропущенные периоды

### 🔐 Приватность и контроль
- 🔒 Все данные хранятся **локально** на вашем компьютере
- 🌐 Поддержка **прокси** (SOCKS5/HTTP) для работы в условиях ограничений
- 🧹 **Автоматическая очистка VRAM** — Whisper и Ollama не конфликтуют за память
- 👤 Доступ только для вашего Telegram-аккаунта (по `TELEGRAM_USER_ID`)

## 📋 Требования

| Компонент | Версия | Зачем |
|-----------|--------|-------|
| **Python** | 3.10+ | Язык выполнения |
| **Telegram Bot Token** | — | Доступ к API Telegram |
| **Ollama** | 0.1.30+ | Локальный запуск LLM |
| **NVIDIA GPU** (опционально) | CUDA 12.x | Ускорение Whisper/LLM |
| **Obsidian** (опционально) | — | Просмотр заметок |

### 🧠 Поддерживаемые модели

| Задача | Рекомендуемая модель | Размер | Требует VRAM |
|--------|---------------------|--------|--------------|
| 🎤 Транскрибация | `large-v3` (Whisper) | ~6 ГБ | 10 ГБ (CUDA) / 0 ГБ (CPU) |
| 🧹 Очистка текста | `llama3:8b-instruct` | ~4.7 ГБ | 6 ГБ |
| 📝 Суммари | `qwen2.5:7b-instruct` | ~4.2 ГБ | 6 ГБ |
| 🤔 Глубокий анализ | `deepseek-r1:8b` | ~5 ГБ | 8 ГБ |

> 💡 **Совет:** Для начала используйте `WHISPER_DEVICE="cpu"` — работает без GPU, просто чуть медленнее.

## ⚙️ Установка

### 1. Клонирование и окружение

```bash
# Клонировать репозиторий (или создать структуру вручную)
git clone https://github.com/yourusername/obrain.git
cd obrain

# Создать виртуальное окружение
python3.12 -m venv .venv
source .venv/bin/activate

# Установить зависимости
pip install -r requirements.txt
```

### 2. Настройка `.env`

Скопируйте `.env.example` в `.env` и заполните:

```bash
cp .env.example .env
nano .env  # или ваш редактор
```

**Минимальная конфигурация:**
```ini
# 🔐 TELEGRAM
BOT_TOKEN="123456:ABC-DEF1234ghIkl-zyx57W2v1u123ew11"
TELEGRAM_USER_ID="7882141240"

# 📂 PATHS (укажите путь к вашему хранилищу Obsidian)
PATH_TO_RAW="/home/user/Obsidian/00_Inbox/raw_data.md"
PATH_TO_JOURNAL="/home/user/Obsidian/01_Daily"
PATH_TO_SUMMARY="/home/user/Obsidian/02_Reviews"

# 🤖 LLM (Ollama)
OLLAMA_BASE_URL="http://127.0.0.1:11434"
OLLAMA_MODEL_CLEAN="llama3:8b-instruct"
OLLAMA_MODEL_SUMMARY="qwen2.5:7b-instruct"

# 🎙️ WHISPER
WHISPER_MODEL="large-v3"
WHISPER_DEVICE="cpu"          # или "cuda", если есть GPU
WHISPER_COMPUTE_TYPE="int8"   # экономит память

# 🌐 PROXY (опционально, для обхода ограничений)
PROXY_TYPE="socks5"
PROXY_HOST="127.0.0.1"
PROXY_PORT="12334"
```

### 3. Скачивание моделей

```bash
# 🔹 Whisper (скачается автоматически при первом запуске)
# Или предзагрузить:
python download_whisper_models.py

# 🔹 Ollama модели
ollama pull llama3:8b-instruct
ollama pull qwen2.5:7b-instruct
```

### 4. Запуск

```bash
# Из корня проекта
python main.py
```

**Ожидаемый вывод:**
```
✅ Конфигурация успешно загружена
🚀 Запуск бота...
🌐 Прокси: socks5h://127.0.0.1:12334
✅ Telegram бот инициализирован
✅ Бот готов к работе
```

### 📤 Отправка заметок

1.  Откройте диалог с вашим ботом в Telegram
2.  Отправьте:
    - 📝 **Текст** — будет очищен и сохранён
    - 🎤 **Голосовое сообщение** — транскрибируется → очищается → сохраняется

## 🔧 Устранение неполадок

### ❌ Бот не отвечает на голосовые

**Симптом:** Текст работает, ГС — тишина.

**Решение:**
1.  Проверьте `content_types` в `handlers.py`:
    ```python
    @bot.message_handler(content_types=['text', 'voice'], func=lambda m: True)
    ```
2.  Убедитесь, что `pysocks` установлен: `pip show pysocks`
3.  Проверьте логи: включите `DEBUG_MODE=True` в `.env`

### ❌ Ошибка `libcublas.so.12 not found`

**Причина:** Whisper пытается использовать CUDA, но библиотека не найдена.

**Решение (быстрое):**
```ini
# В .env:
WHISPER_DEVICE="cpu"
WHISPER_COMPUTE_TYPE="int8"
```

**Решение (полное):** Установите CUDA 12.x Toolkit параллельно с вашим драйвером.

### ❌ `ImportError: attempted relative import`

**Причина:** Относительные импорты не работают при прямом запуске.

**Решение:**
- Используйте абсолютные импорты: `from Obrain.config import settings`
- Запускайте из корня проекта: `python main.py`, а не `python schedulers/weekly.py`
- Убедитесь, что в `schedulers/__init__.py` файл существует (может быть пустым)

### ❌ `ValidationError: Field required` при загрузке конфига

**Причина:** `.env` не найден или пуст.

**Решение:**
- Проверьте, что файл `.env` лежит в корне проекта
- Убедитесь, что в `config.py` используется абсолютный путь:
  ```python
  ENV_FILE = Path(__file__).resolve().parent / ".env"
  load_dotenv(dotenv_path=ENV_FILE)
  ```

### ❌ Telegram замедлен, бот «виснет»

**Решение:**
1.  Настройте прокси в `.env` (см. раздел выше)
2.  Увеличьте таймауты:
    ```ini
    TELEGRAM_TIMEOUT="60"
    TELEGRAM_LONG_POLLING_TIMEOUT="120"
    ```
3.  Убедитесь, что `use_threads=False` в `bot.py` (повышает стабильность)

### ❌ Ollama не отвечает

**Проверка:**
```bash
# Проверьте, что Ollama запущен
curl http://127.0.0.1:11434/api/tags

# Проверьте доступность модели
ollama list

# Протестируйте запрос вручную
curl http://127.0.0.1:11434/api/generate -d '{
  "model": "llama3:8b-instruct",
  "prompt": "Привет",
  "stream": false
}'
```
